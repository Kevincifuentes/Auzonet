FAQ : Preguntas frecuentes

¿Qué es Auzonet?
Auzonet es una plataforma orientada a formar un ambiente vecinal, especificamente orientado a la ciudad de Bilbao, aunque con la idea de ampliar el área de uso a más ciudades. Lo que se pretende es compartir aquellas cosas de las que disponemos en nuestras casa pero que no las utilizamos o ya no las queremos. A modo de darles una nueva vida, se cuelgan en esta web y el que las necesite contacta con el propietario. ¡Así de simple!
¿Cómo funciona?
Después de un registro, podremos acceder a todo tipos de objetos disponibles en el sistema y que podemos pedir para que nos los presten otros usuarios. Con contactar con el propietario mediante email o teléfono, podemos llegar a un trato para que nos deje un determinado objeto (o nos lo venda).
¿Qué debo hacer para participar?
Unicamente tenemos que crearnos una cuenta a nuestro nombre en el sistema y tendremos acceso completo a todas las funcionalidades, con las que podremos encontrar miles de objetos en proceso de prestamo. Para registrarse será necesario un email válido. Por otro lado, nosotros podemos añadir nuestro propios objetos, mediante el formulario correspondiente y añadiendo los datos del objeto y una foto identificativa.
¿Cómo encuentro lo que necesito?
Auzonet utiliza las últimas tecnologías con un buscador de objetos óptimizado para buscar los objetos que se necesiten rápidamente, ya sea por nombre, propietario o cualquier otro parámetro. Adicionalmente se dispone de un mapa para encontrar los objetos que tengamos más cerca geograficamente.
¿Cómo contacto con el propietario?
Para contactar con el propietario se dispondrá del email y el teléfono que ha facilitado el propio propietario para dicho objeto, ya sea fijo o móvil.
